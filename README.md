## Installation & Usage
### Prerequisites
Ensure you have **Node.js** and **npm** installed.

### Clone the Repository
```sh
git clone https://github.com/ayeshakhalil0/todo-list.git
cd todo-list
```

### Install Dependencies
```sh
npm install
```

### Run the App
```sh
npm run dev
```
